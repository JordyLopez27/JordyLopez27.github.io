Course description

This course introduces computational techniques in algebraic geometry: Gröbner bases, elimination, resultants, primary decomposition, toric varieties and tropical methods, numerical algebraic geometry, and applications. Students will learn both symbolic and numerical tools (Macaulay2, Singular, Bertini, Gfan, polymake, SageMath) and apply them to research-type problems.

Learning outcomes

  • Use Gröbner bases to solve polynomial systems and perform elimination.
  • Understand computational approaches to schemes and varieties: primary decomposition, dimension, degree.
  • Apply numerical methods (homotopy continuation) to compute solution sets.
  • Use software packages (Macaulay2, Singular, Gfan, Bertini) effectively for experiments and research.
  • Read and present short research papers in computational algebraic geometry.

Prerequisites

Basic algebra (graduate algebra or equivalent), linear algebra, and familiarity with programming (Python/command line) recommended.

Assignments & projects

There will be regular problem sets with symbolic and computational parts. Two short computational projects (midterm-style) and a final project are required. Example project ideas:

  • Compute tropicalization of a family of plane curves and compare with Newton polytope predictions.
  • Use homotopy continuation to solve a parametrized family of kinematic equations and analyse solution counts.
  • Implement an algorithm to compute primary decomposition for special classes of ideals and test on examples.

All code should be version-controlled and reproducible; include README with instructions to run experiments. You may use GitHub/GitLab for submission and a short PDF report (5–8 pages).

Recommended software

  • Macaulay2 — symbolic computations in algebraic geometry. (https://faculty.math.illinois.edu/Macaulay2/)
  • Singular — Gröbner bases and polynomial computations. (https://www.singular.uni-kl.de/)
  • Bertini — numerical algebraic geometry / homotopy continuation. (https://bertini.nd.edu/)
  • Gfan — tropical computations and Gröbner fans. (https://users-math.au.dk/jensen/software/gfan/)
  • polymake — polyhedral computations used in tropical/toric work. (https://polymake.org/)
  • SageMath — a Python-based environment that aggregates many tools. (https://www.sagemath.org/)

If you have trouble installing software, bring your laptop to office hours or use the provided lab machines. We will publish a step-by-step installer guide and example notebooks early in the semester.

Primary & suggested reading

  • Cox, Little, O'Shea — Ideals, Varieties, and Algorithms (for Gröbner bases and computational algebra).
  • D. Eisenbud — Commutative Algebra with a View Toward Algebraic Geometry (reference for algebraic foundations).
  • B. Sturmfels — Solving Systems of Polynomial Equations and Gröbner Bases and Convex Polytopes (combinatorial and computational perspective).
  • Software manuals and selected research papers (will be posted on the course site / repository).
